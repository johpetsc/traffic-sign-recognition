The German Traffic Sign Recognition Benchmark

Example code for Python

**********************************************

This archive contains example code for MATLAB for 
reading the datasets and corresponding annotations.
It contains the following files:

    Readme.txt           - This file
    readTrafficSigns.py  - Function to read the training dataset

This python script depends on matplotlib to read the image files.


**********************************************
readTrafficSigns.py 
**********************************************

The file readTrafficSigns.py provides the following function:

    readTrafficSigns(rootpath):

        Reads traffic sign data for German Traffic Sign Recognition Benchmark.

        Arguments: path to the traffic sign data, for example './GTSRB/Training'

        Returns:   list of images, list of corresponding labels
